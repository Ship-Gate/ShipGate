# IntentOS Tutorials

Welcome to IntentOS tutorials! These step-by-step guides will help you learn how to use IntentOS to specify, implement, verify, and gate your code.

## Available Tutorials

1. **[Hello World](./01-hello-world.md)** - Your first IntentOS project (spec + implementation + verify + gate)
2. **[REST API](./02-rest-api.md)** - Building a REST API with IntentOS specifications
3. **[Authentication](./03-authentication.md)** - Implementing secure authentication flows
4. **[Property-Based Testing](./04-property-based-testing.md)** - Using PBT to find edge cases
5. **[Chaos Testing](./05-chaos-testing.md)** - Testing resilience with chaos engineering

## Prerequisites

- Node.js 18+ installed
- Basic knowledge of TypeScript/JavaScript
- Terminal/command line familiarity

## Quick Start

Each tutorial includes:
- ✅ Exact commands to run
- ✅ Expected outputs
- ✅ Troubleshooting section
- ✅ Complete sample code

Start with [Hello World](./01-hello-world.md) to get up and running in under 30 minutes!

## Sample Projects

Each tutorial has a corresponding sample project in `/samples/tutorials/`:
- `hello-world/` - Complete Hello World example
- `rest-api/` - REST API example
- `auth/` - Authentication example
- `pbt/` - Property-based testing example
- `chaos/` - Chaos testing example

You can run these samples directly or follow along with the tutorials to build them yourself.
