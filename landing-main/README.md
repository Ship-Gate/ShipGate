  # Landing page design

  This is a code bundle for Landing page design. The original project is available at https://www.figma.com/design/kouepiu7sU3hLytbCrvWKO/Landing-page-design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deployment

  Run `npm run build` to build the production version.

  Run `npm run deploy` to deploy the production version to the server.
