## [0.1.0] - 2024-01-31

### Added
- Initial release of ISL packages
- `@isl-lang/parser` - ISL recursive descent parser
- `@isl-lang/typechecker` - Semantic analyzer
- `@isl-lang/evaluator` - Expression evaluator
- `@isl-lang/codegen` - Code generators (umbrella package)
- `@isl-lang/cli` - Command-line interface
- `@isl-lang/repl` - Interactive REPL
- `@isl-lang/lsp-server` - Language Server Protocol implementation
- `@isl-lang/lsp-core` - Core LSP intelligence
- VS Code extension for ISL support
